It also offers all the methods and properties of the object returned by \doxylink{md__project5___h_r_2_project5___h_r___front___end_2node__modules_2fastq_2_r_e_a_d_m_e_fastqueue}{\`{}fastqueue\`{}} with the modified \doxylink{md__project5___h_r_2_project5___h_r___front___end_2node__modules_2fastq_2_r_e_a_d_m_e_pushPromise}{\`{}push\`{}} and \doxylink{md__project5___h_r_2_project5___h_r___front___end_2node__modules_2fastq_2_r_e_a_d_m_e_unshiftPromise}{\`{}unshift\`{}} methods.

Node v10+ is required to use the promisified version.

Arguments\+:
\begin{DoxyItemize}
\item {\ttfamily that}, optional context of the {\ttfamily worker} function.
\item {\ttfamily worker}, worker function, it would be called with {\ttfamily that} as {\ttfamily this}, if that is specified. It MUST return a {\ttfamily Promise}.
\item {\ttfamily concurrency}, number of concurrent tasks that could be executed in parallel.
\end{DoxyItemize}
