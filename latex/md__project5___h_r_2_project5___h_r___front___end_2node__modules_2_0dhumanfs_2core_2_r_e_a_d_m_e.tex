The core functionality for humanfs that is shared across all implementations for all runtimes. The contents of this package are intentionally runtime agnostic and are not intended to be used alone.

Currently, this package simply exports the {\ttfamily Hfs} class, which is an abstract base class intended to be inherited from in runtime-\/specific hfs packages (like {\ttfamily @humanfs/node}).

\mbox{[}!\+WARNING\mbox{]} This project is {\bfseries{experimental}} and may change significantly before v1.\+0.\+0. Use at your own caution and definitely not in production! \end{quote}

The {\ttfamily Hfs} class contains all of the basic functionality for an {\ttfamily Hfs} instance {\itshape without} a predefined impl. This class is mostly used for creating runtime-\/specific impls, such as {\ttfamily Node\+Hfs} and {\ttfamily Deno\+Hfs}.

You can create your own instance by providing an {\ttfamily impl} directly\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ hfs\ =\ new\ Hfs(\{\ impl:\ \{\ async\ text()\ \{\}\ \}\});}

\end{DoxyCode}


The specified {\ttfamily impl} becomes the base impl for the instance, meaning you can always reset back to it using {\ttfamily reset\+Impl()}.

You can also inherit from {\ttfamily Hfs} to create your own class with a preconfigured impl, such as\+:
