This usage is equivalent to calling {\ttfamily use\+Loader\+Data} and {\ttfamily use\+Action\+Data}, but consolidates all route data access into one hook\+: {\ttfamily use\+Route}.

Note\+: when calling {\ttfamily use\+Route()} (without a route ID), TS has no way to know which route is the current route. As a result, {\ttfamily loader\+Data} and {\ttfamily action\+Data} are typed as {\ttfamily unknown}.
