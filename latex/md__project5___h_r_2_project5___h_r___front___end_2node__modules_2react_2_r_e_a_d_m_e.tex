 React is a Java\+Script library for creating user interfaces.

The {\ttfamily react} package contains only the functionality necessary to define React components. It is typically used together with a React renderer like {\ttfamily react-\/dom} for the web, or {\ttfamily react-\/native} for the native environments.

{\bfseries{Note\+:}} by default, React will be in development mode. The development version includes extra warnings about common mistakes, whereas the production version includes extra performance optimizations and strips all error messages.
